Save bot token to DISCORD_TOKEN env var with 'export DISCORD_TOKEN="token"', make sure the bot has perms in the server/channel it will operate in

Save discord channel ID to DISCORD_CHANNEL env var with 'export DISCORD_CHANNEL="uint64 channel id"'

To install the dpp lib visit: https://dpp.dev/md_docpages_01_installing.html

To get information on how to create the bot for this program: https://discordpy.readthedocs.io/en/stable/discord.html
